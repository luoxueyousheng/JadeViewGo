窗口拖动区（jade-region-drag）

按住这里拖动窗口 · 双击最大化/还原 · 右键只是普通页面右键

HTML 布尔属性即生效（运行时自动注入，无需 JS）：jade-region-drag 标记容器及后代可拖动， jade-region-no-drag 在其中挖洞保持可点击。与旧 CSS -webkit-app-region: drag 的区别：右键不弹系统标题栏菜单。顶部标题栏也已换用此方式。
